My Projects

1. Metasurface Inspired Printed Monopole Antenna for Efficient Energy Harvesting System

Major project done as a part of undergraduate program. Monopole antenna is designed using CST studio suite. The antenna is included with meta-material superstrate to enhance the performance of antenna in terms of efficiency, power consumption, gain.

2. Smart Driving Assistance

Mini project based on Internet of Things (IoT). Smart driving assistance system is designed using Arduino IDE. Arduino UNO, HC-05 bluetooth Module, Ultrasonic motion sensor are the major hardware components used. The system send warning signal to the driver based on the distance of objects infront of the vehicle.

3. Obstacle Avoiding Robot

Mini project based on Internet of Things (IoT). The project is designed using Arduino IDE tool, Ultrasonic sensors, Servo motor, Arduino UNO board. The robot designed is self controlled by detecting obstracles infront of it and automatically changes its direction to avoid collision.
